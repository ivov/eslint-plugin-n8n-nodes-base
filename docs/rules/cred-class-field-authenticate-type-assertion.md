# cred-class-field-authenticate-type-assertion

In a credential class, the field `authenticate` must be typed `IAuthenticateGeneric`

üìã This rule is part of the `plugin:n8n-nodes-base/credentials` config.

üîß Run ESLint with `--fix` option to autofix the issue flagged by this rule.

## Examples

‚ùå Example of **incorrect** code:

```js

    displayName = 'My Test API';
    documentationUrl = 'myTest';
    authenticate = {
        type: 'generic',
        properties: {
            headers: {

            },
        },
    } as IAuthenticateGeneric;

```

    documentationUrl = 'myTest';

        type: 'generic',
        properties: {
            headers: {
                Authorization: '=Token {{$credentials.token}}',
            },
        },
    };
